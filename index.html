EasyZoom jQuery image zoom plugin

EasyZoom is an elegant jQuery image zoom plugin based on the original work by Alen Grakalic. EasyZoom has been extended with public methods, optimised performance and includes an optional extension to create an image gallery and works on Apple iOS touch enabled devices such as the iPad.
